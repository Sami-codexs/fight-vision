This project analyzes UFC fight videos using computer vision.

Day 1:
- Video loading
- Frame-by-frame processing
- Timestamp calculation

Day 2:
- Octagon ROI extraction
- Person detection using YOLOv8

Day 3:
- IOU-based fighter tracking
- Stable fighter identities (A / B)

Future work:
- Stance detection
- Strike detection
- Scoring logic
